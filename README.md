##百度前端技术学院简介
百度前端技术学院诞生于2014年，由百度最大规模的前端技术组织 EFE 团队发起成立。
目前学院是由百度校园品牌部、百度多模交互搜索部以及百度校园招聘组联合组织的，面向大学生人群，免费的一个技术学习、交流与分享平台。</br>
